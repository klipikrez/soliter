This package contains simple common uis that was not implemented by default unity ui

-> NestedScrollRect:
   ScrollRect version that support send drag event to parent in axis not captured by this current object.

-> MaxLayoutElement:
   LayoutElement that support MaxWidth/MinWidth

-> LinearLayoutGroup:
   Vertical or Horizontal LayoutGroup. Just check the bool to apply one, or another

-> FlexLayoutGroup:
   Unity Implementation of Android Flexbox. Can overflow elements that does not fit in a row/collum
